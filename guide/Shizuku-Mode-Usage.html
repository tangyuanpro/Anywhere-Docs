# Shizuku 模式

# 适用场景

同 Root 模式

# 为什么选择 Shizuku 模式？

Shizuku 可以让你的应用直接使用系统 API，使用户无需向 Anywhere- 授予 Root 权限也可以在手机上执行 adb 指令。只需将手机连接电脑，执行一条 adb 指令即可激活 Shizuku。有关 Shizuku Manager 的激活教程请前往 Shizuku Manager 官方文档查看。

注意

无需 Root 权限不代表手机可以不具有 Root 权限，因打开 Activity 的 adb 指令需要 Root 权限。Shizuku 模式只是让本程序可以不获取 Root 权限，从而保证安全、提高性能并且便于管理。
